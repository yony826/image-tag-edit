# 图片标签处理工具 - 开发文档

## 1. 项目概述

### 1.1 项目目标
开发一个图片标签处理工具，通过本地部署的 Qwen3.5:9B 大模型识别图片内容并自动打上标签，支持多标签搜索功能。

### 1.2 核心技术方案
| 技术选型 | 说明 |
|---------|------|
| 编程语言 | Python 3.10+ |
| 桌面框架 | PyQt5 |
| 大模型调用 | LM Studio HTTP API |
| 数据存储 | JSON 文件（每个图片文件夹一个 metadata.json） |

---

## 2. 功能需求

### 2.1 核心功能

#### 2.1.1 图片导入
- 支持选择文件夹批量导入图片
- 支持拖拽导入图片
- 支持的图片格式：JPG、JPEG、PNG、BMP、GIF、WebP

#### 2.1.2 标签生成
- 调用本地 LM Studio API 加载 Qwen3.5:9B 模型
- 对每张图片进行内容识别
- 生成 3-8 个标签（标签为中文或英文名词）
- 标签存储在图片所在文件夹的 `metadata.json` 文件中

#### 2.1.3 标签搜索
- 支持按标签搜索图片
- 支持 AND（同时满足多个标签）组合搜索
- 支持 OR（满足任一标签）组合搜索
- 搜索结果实时显示

#### 2.1.4 图片浏览
- 以网格形式展示图片
- 点击图片可查看大图和标签信息
- 支持图片预览放大

### 2.2 用户交互流程

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  导入图片   │ ──▶ │  AI识别标签 │ ──▶ │  标签存储   │ ──▶ │  标签搜索   │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

### 3.2 数据结构

#### metadata.json 结构
```json
{
  "images": [
    {
      "filename": "photo1.jpg",
      "tags": ["烟花", "夜空", "庆典", "火花", "红色"],
      "processed_at": "2026-03-05 10:30:00",
      "model": "qwen3.5-9b"
    }
  ]
}
```

### 3.3 LM Studio API 调用

**请求地址**: `http://localhost:1234/v1/chat/completions`

**请求示例**:
```json
{
  "model": "qwen3.5-9b",
  "messages": [
    {
      "role": "user",
      "content": "请描述这张图片的内容，并给出5-8个关键词标签。直接返回标签，用逗号分隔，不要其他描述。"
    }
  ],
  "stream": false
}
```
